# Discovery Use Case Diagram

## Overview
This diagram describes the tour discovery and browsing functionality for the Mountain Tours System. These features are available to all users without authentication.

## Actors

| Actor | Description |
|-------|-------------|
| **User** | Any visitor to the platform (authenticated or not) |

## Use Cases

### Tour Discovery
- **Search Tours** - User searches for tours using filters (location, date, difficulty, price, etc.)
- **Browse Tours** - User explores available tours through categories or listings
- **View Tour Details** - User views comprehensive information about a specific tour

### Profile Viewing
- **View Guide Profile** - User views a guide's profile including bio, certifications, and experience
- **View Tour Operator Profile** - User views a tour operator's company profile and offerings

### Reviews & Ratings
- **Read Reviews & Ratings** - User reads feedback and ratings from previous tour participants

## Notes
- All use cases in this diagram are accessible without authentication
- This enables potential customers to explore offerings before registering
- Discovery features help users make informed decisions about bookings
